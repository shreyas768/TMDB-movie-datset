Introduction¶

In this Data Analysis, we'll analysis a TMDB movies database file which has collected information of about 10 thousand movies which include their budget, release date, cast, director etc. The data, as provided by Udacity, is taken from below mentioned link.

https://www.google.com/url?q=https://d17h27t6h515a5.cloudfront.net/topher/2017/October/59dd1c4c_tmdb-movies/tmdb-movies.csv&sa=D&ust=1532469042115000

Questions that can be formulated and answered using this dataset are:-¶
1 Which movie made the most profit?¶
2 Which movie made the least profit (or most loss) ?¶
3 Which is the longest and shortest movie?¶
4 Most and least expensive Movie?¶
5 Which movie earned most and least revenue?¶
6 Least and most popular movie?¶
7 Genre in which most number of movies are made?¶
8 Movies of which genre genreate more profit and require more budget? How the trend of number of movies made shift accordingly?¶
9 In which year, all the movies together made the most profit?¶
10 How is the rise in profit each year, taking all the movies together?¶
11 Which year saw most number releases?¶
12 Actors who have appeared in atleast 50 movies?¶
13 Actors who made most appearances since 1990,2000 and 2010 ?¶
14 Trend in runtime of movies released after 1990?¶
15 Month that saw highest number of releases in all of the years?¶
16 Monthwise (of all the years ) trend of the profit made?¶
17 Average runtime of movies released after 1990?¶
18 Directors with most number of movies ?¶
19 Directors with most profitable/successful movies, earning minimum 1 billion USD (since 1990) ?¶
20 Correlation among various variables of the dataset.¶

8. Now let's analyse which movie genre has generated more profit and how many movies are made for each profit bracket.¶

We'll create a function 'Above', in which we'll put the profit amount as per our choice. The output will be a graph showing number of movies made in each genre above the profit amount we've entered.

In [1079]:
def Above(Profit_value):
    profit = tmdb[tmdb['profit(in_USD)'] > Profit_value ]
    genre = profit['genres'].str.cat(sep = '|')
    genre = pd.Series(genre.split('|'))
    count = genre.value_counts(ascending = False)
    plot = count.plot.barh(color = 'red', fontsize = 13)
    plot.set(title = 'Genres in Profitable Movies')
    plot.set_xlabel('Number of Movies', color = 'black', fontsize = '13')
    plot.figure.set_size_inches(12,4.5)
    return plot

9. We'll do similar analysis on revenue column. def BudgetQuant(q):
    Qbudget = tmdb['budget(in_USD)'].quantile(q) 
    budget = tmdb[tmdb['budget(in_USD)'] > Qbudget ]
    genre2 = budget['genres'].str.cat(sep = '|')
    genre2 = pd.Series(genre2.split('|'))
    count2 = genre2.value_counts(ascending = False)
    plot2 = count2.plot.barh(color = 'red', fontsize = 13)
    plot2.set(title = 'Genres in Movies with most budget')
    plot2.set_xlabel('Number of Movies', color = 'black', fontsize = '13')
    plot2.figure.set_size_inches(12,4.5)
    return plot2

Conclusion :-¶

Thus, after detailed analysis of this dataset, we can concluded that, for a movie to be successful:¶
A. It should have a run time of approx 110 minutes.¶
B. It should be under the genre: drama, comedy, thriller etc, if it is a low-budget movie.¶
C. It should be under the genre: Action, Adventure etc, if it is a high-budget movie.¶
D. It should release during May, June, November or December.¶
E. If it is a short movie, animation is the preferred genre.¶

Notes and Limitations :-

  1. Many other conclusions have been drawn and mentioned during analysis itself and thus, are not repeated here.
  2. It is not guaranteed that this formula is gonna work for a successful movie, but it shows us that we have high probability of making high profits if we had similar characteristics as such. If the movie is worth, people's high expectations would lead to biased results thus resulting in high profits. We see this in real life, especially in sequels.
  3. The budget and revenue columns did not have a currency specified and hence we've taken USD for all our monetary analysis.
  4. Usual practice is to drop the rows with NaN values, but we've kept them so as to make our analysis as detailed as possible.
  5. Attracting people towards a movie ultimately comes down to story mainly and many other important influential factors.
